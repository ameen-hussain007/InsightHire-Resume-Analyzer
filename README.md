InsightHire is a web-based application that analyzes resumes and matches them with suitable job roles using natural language processing techniques.

## Overview
The application allows users to upload a resume in PDF format and receive insights such as extracted skills, suggested job roles, and a matching score.

## Features
- Resume upload and parsing
- Skill extraction using NLP
- Job role matching with relevance score
- Clean and simple user interface

## Technology Stack
- Backend: Python, Flask
- NLP: spaCy
- Frontend: HTML, CSS, JavaScript
